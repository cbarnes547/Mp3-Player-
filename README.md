# Mp3-Player
# Project Overview

I developed a functional mock MP3 player application using JavaFX that focuses on managing data through parallel arrays. The core logic involves synchronizing song titles and file paths across multiple arrays, ensuring data integrity during sorting and update operations. I implemented event-handling methods within a Controller class to respond to user interactions, such as adding songs and persisting playlists to text files. The project utilizes a ListView control and ObservableList to bridge the gap between back-end data storage and the graphical user interface. By building this system, I demonstrated proficiency in file I/O, selection-based sorting algorithms, and the MVC (Model-View-Controller) design pattern.
